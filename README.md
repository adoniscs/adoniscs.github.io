# Projeto To Do em Javascript

Essa pequena aplicação web permite adicionar e excluir tarefas. Além disso, ele foi construido com um local storage que permite salvar as tarefas no browser, com isso, se você quiser continuar depois a fazer as tarefas, ela continuarão lá =) 

 Esse projeto foi basedo no curso da [Rocketseat Starter](https://rocketseat.com.br/starter "Rocketseat Starter"). Em um dos módulos é feito o To Do bem simples, mas funcional. Isso me levou a evoluir o layout e deixando mais visual:
